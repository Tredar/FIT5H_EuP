Definition
	Array ist eine geordnete Liste, die Verweise auf andere Objekte enthält.

Neues Array anlegen
	seats = []
	seats = Array.new

	seats = ["wert1", "wert2"]
	seats = %w(wert1 wert2)

Werte hinzufügen
	seats << "wert3"
	seats[3] = "wert4"
	seats.push("wert5")

Methoden von Arrays
	Array Größe:
		seats.size
	letztes Element entfernen
		seats.pop
	Array leer
		seats.empty?
	Löschen von Element
		seats.delete_at(3)


Iterationa-Methoden:
var = %w(ich u du)
var.each
	Methode zum durchlafen der einzelnen Elemente

var.select
	Methode, die im Block einen Boolischen Wert erwartet und bei True den Inhalt zurückgibt
		z.B. ret = var.select {|v| v.size > 1}
		ret enthält ['ich', 'du']

var.reject
	Gegenteil von select
	ret = var.reject {|v| v.size > 1}
	ret enthält ['u']

var.sort
	gibt ein sortiertes Array zurück
